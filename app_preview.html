ColorCard Composable Implementation

This demonstrates a Jetpack Compose ColorCard function with three different modifier combinations:

Red Card (Standard ColorCard function):

.size(200dp × 150dp) → .padding(16dp) → .background(Red) → .border(3dp, Black)

Blue Card (Alternative combination):

.size(180dp × 120dp) → .background(Blue) → .border(4dp, White) → .padding(12dp)

Green Card (Third combination):

.padding(24dp) → .size(220dp × 140dp) → .border(2dp, Yellow) → .background(Green)

Requirements fulfilled:

  • ✅ ColorCard composable function with Color and String parameters
  • ✅ Three different colors displayed (Red, Blue, Green)
  • ✅ Centered labels in each card
  • ✅ Uses all four required modifiers: padding, background, border, size
  • ✅ At least two different modifier combinations demonstrated
